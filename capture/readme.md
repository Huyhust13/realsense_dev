**Capture**
===========
Usage:

Buil program: Open terminal
```bash
cd <...>/capture/
mkdr build && cd build
cmake ..
make
```  

Run:
```bash
./capture <mode>
```

* `<mode>:color` show color image 
* `<mode>:depth` show depth image 
* `<mode>:infrared` show infrared image 

Press `Esc` for quit.
